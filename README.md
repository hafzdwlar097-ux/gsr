
# تطبيق جسور (Bridges P2P) - مفتوح المصدر

تطبيق إغاثة مجتمعي يعمل بالكامل بدون إنترنت عبر الشبكة المحلية.

## المميزات
- **بدون إنترنت:** مزامنة البلاغات بين الأجهزة القريبة عبر المتصفح.
- **خدمات طوارئ:** (شرطة، إسعاف، مطافئ، أمن).
- **كاميرا:** إمكانية التقاط صور للبلاغات.
- **خصوصية:** يعمل محلياً بالكامل ولا يرسل بيانات لأي سيرفر خارجي.

## كيف ترفع التطبيق على GitHub؟
1. قم بإنشاء مستودع جديد (Repository) على GitHub.
2. انسخ جميع ملفات المشروع في مجلد واحد على جهازك.
3. ارفع الملفات للمستودع.
4. فعل خاصية **GitHub Pages** من الإعدادات ليعمل التطبيق كـ رابط مباشر.

## كيف تحصل على ملف ZIP؟
- ببساطة، قم بضغط مجلد المشروع بالكامل في ملف واحد بصيغة `.zip`. يمكنك إرسال هذا الملف لأي شخص وسيعمل التطبيق بمجرد فتح `index.html`.
